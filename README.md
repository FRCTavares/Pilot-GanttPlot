# Pilot-GanttPlot

A small Python-generated HTML roadmap for visualising a pilot career timeline.

## Features

- IFA training phases
- Airline application and entry phases
- First Officer development phases
- Dynamic today marker
- Possible paid airline phase marker
- Age and year reference rows
- Light/dark mode toggle

## Files

- build_pilot_roadmap.py: source file used to generate the roadmap
- pilot_roadmap.html: generated roadmap, open this in a browser

## Usage

Generate the roadmap:

    python3 build_pilot_roadmap.py

Open it:

    open pilot_roadmap.html

## Editing

Change dates, labels, phases, milestones, or colours inside:

    build_pilot_roadmap.py

Then regenerate:

    python3 build_pilot_roadmap.py

## Notes

This roadmap is a planning estimate, not a guarantee. Airline selection, type rating, line training, and the first paid phase depend on hiring market conditions and airline contract terms.
